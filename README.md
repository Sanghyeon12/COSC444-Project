# COSC444 Project

DNA / Protein motif scanner using regular expressions and FASTA input.

This tool loads motifs from JSON, converts them to regex, scans sequences from FASTA files, and reports all matched positions. It also includes simple baseline and evaluation modules.

## Layout

- `src/io/` : FASTA loading and alphabet checks  
- `src/motifs/` : motif loading and IUPAC â†’ regex conversion  
- `src/scan/` : scanner functions, reporting, and CLI main  
- `src/eval/` : randomization baseline and simple metrics  
- `motifs/` : JSON motif definitions for DNA and protein  
- `data/` : small demo FASTA file  
- `out/` : output folder for results generated by the program  
- `tests/` : pytest unit tests
