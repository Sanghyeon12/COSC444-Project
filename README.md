# COSC444-Project

DNA / Protein motif scanner using regular expressions and FASTA input.

## Layout

- `src/io`: FASTA + alphabet handling
- `src/motifs`: motif definitions + IUPAC handling
- `src/scan`: scanner + reporting + CLI main
- `src/eval`: randomization baseline + metrics
- `motifs/`: JSON motif definitions for DNA and protein
- `data/`: small demo FASTA data
- `out/`: outputs generated by the program
- `tests/`: unit tests
